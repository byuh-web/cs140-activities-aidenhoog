1. List out the different ways to select an item

Many different ways to select an item. Using class ID and element selectors.

2. Explain the difference between class and ID selectors

Class selectors can be used on multiple elements where ID selectors are unique to one element.

3. What is the difference between a SPAN tag and a DIV tag?

A SPAN tag is an inline element while a DIV tag is a block-level element.

4. What is the difference between a DIV tag and a HEADER, MAIN, or FOOTER tag?

DIV tags are generic containers while HEADER, MAIN, and FOOTER tags have specific meanings for structuring a webpage.

5. What are the different ways to select a color?

Colors can be selected using color names, HEX codes, RGBA values.
